Descrição:

Este é um jogo simples da cobrinha implementado em Python usando a biblioteca Pygame. A cobrinha é controlada pelo jogador para comer a comida (pontos vermelhos) enquanto evita colidir com as paredes ou com o próprio corpo. O jogo termina quando a cobrinha colide com uma parede ou com seu próprio corpo. A cobrinha cresce ao comer a comida, aumentando o desafio. A interface gráfica é desenhada na tela, e o jogador pode usar as setas do teclado para mudar a direção da cobrinha.
