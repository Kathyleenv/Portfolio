Descrição:

Este projeto visa automatizar o processo de login e cadastro de produtos no sistema da empresa utilizando a biblioteca pyautogui para interações com a interface do usuário. Os dados dos produtos são importados de uma tabela CSV com a ajuda do pandas. Cada produto é cadastrado individualmente no sistema, preenchendo os campos necessários e enviando o formulário automaticamente. O objetivo é aumentar a eficiência e reduzir o tempo necessário para realizar cadastros manuais repetitivos.
